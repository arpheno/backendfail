Being a web framework, Django needs a convenient way to generate HTML dynamically.

The most common approach relies on templates.

A template contains the static parts of the desired HTML output as well as some special syntax describing how dynamic content will be inserted. For example it can contain dynamically generated links to the rest of the application:

Create new blog! View blogs {% endblock %}
